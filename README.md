# Lumina - Smart Lamp Dashboard

A 3D, interactive tracking dashboard for cryptocurrency prices, built with Three.js and the CoinGecko API.

## Features
- **3D Visualization**: Interactive glowing sphere that reacts to market trends.
- **Real-time Data**: Fetches live prices for BTC, ETH, and SOL.
- **Cyberpunk Aesthetic**: Glassmorphism, neon accents, and dark mode.
- **Responsive**: Fully optimized for mobile and desktop.

## How to Deploy on GitHub Pages

1. **Create a Repository**:
   - Go to GitHub and create a new repository (e.g., `lumina-lamp`).
   - Initialize it with a README.

2. **Upload Files**:
   - Upload `index.html` to the main branch (main/master).

3. **Enable GitHub Pages**:
   - Go to **Settings** > **Pages**.
   - Under **Build and deployment**, select **Source** as `Deploy from a branch`.
   - Select the branch (e.g., `main`) and folder (`/root`).
   - Click **Save**.

4. **Visit your Site**:
   - GitHub will give you a link (e.g., `https://yourusername.github.io/lumina-lamp/`).
   - Wait a minute for it to build, and you're live!
